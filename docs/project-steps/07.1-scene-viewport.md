# Sous-étape 07.1 — Scene + viewport + grille + échelle

## 1. Objectif utilisateur

Le canvas SVG s'affiche dans la zone principale de l'AppShell. Une grille fine est visible en fond (même sans plan ni pièces). Une barre d'échelle indique la dimension réelle courante (ex. "1 m"). Le canvas est centré sur l'origine monde à l'ouverture.

## 2. Dépendances

- Jalon 06 ✅

## 3. Fichiers à créer / modifier

**Créer :**
- `src/hooks/useViewport.ts` — hook partagé, **état local (useState / useReducer)**, pas dans le store Redux ni dans IDB. Le viewport est volatile par onglet.
  - Expose : `viewport: { zoom, panX, panY }`, `pan(dx, dy)`, `zoomAt(x, y, factor)`, `setViewport(v)`.
  - Expose aussi `worldToScreen(p)` et `screenToWorld(p)` (pures).
  - Le hook est instancié **une seule fois** au niveau `<Scene>` et passe via contexte si plusieurs composants du canvas en ont besoin.
- `src/components/canvas/Scene/ViewportContext.ts` — contexte React fournissant la valeur du hook aux enfants du canvas.
- `src/components/canvas/Scene/index.tsx` — racine SVG. Structure :
  - `<svg width="100%" height="100%">`
  - `<defs>` avec un `pattern` pour la grille.
  - `<rect fill="url(#grid)" />` plein canvas.
  - `<g transform="translate(panX, panY) scale(zoom)">` pour les éléments monde (vide à ce stade — enfants passés via `children`).
  - `<ScaleBar />` overlay non-transformé (coin bas-droite).
- `src/components/canvas/Scene/Scene.module.css` — fond `--canvas-bg`, grille `--grid-color`.
- `src/components/canvas/Scene/Grid/index.tsx` — définition `pattern` de la grille.
- `src/components/canvas/Scene/ScaleBar/index.tsx` — barre d'échelle :
  - Affiche la valeur entière la plus proche en cm/m pour une largeur écran cible (~`6rem`).
  - Choix de la valeur : `pickNiceScale(targetWorldSpan)` — retourne une valeur parmi `{1, 2, 5, 10, 20, 50}` cm ou m selon ordre de grandeur.
  - Trait + tick aux extrémités + label "X cm" ou "X m".
- `src/components/canvas/Scene/ScaleBar/ScaleBar.module.css` — positionné absolute coin bas-droite via tokens `rem`.
- `src/core/viewport.ts` — fonctions pures `worldToScreen`, `screenToWorld`, `pickNiceScale`.
- `src/core/__tests__/viewport.test.ts`.

**Modifier :**
- `src/store/uiSlice.ts` — **ne PAS ajouter** `viewport`. Si présent depuis une itération précédente, le retirer.
- `src/components/layout/AppShell/index.tsx` — monter `<Scene />` dans la zone canvas quand un projet est courant (avec la garantie 06.2 : projet toujours présent).
