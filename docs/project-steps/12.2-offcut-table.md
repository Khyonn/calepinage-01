# Sous-étape 12.2 — Tableau des liens de réutilisation

## 1. Objectif utilisateur

Dans le drawer droit (après le tableau de résumé), l'utilisateur voit la liste des liens de réutilisation : chaque ligne indique la rangée source (chute générée), la rangée cible (chute consommée), la longueur réutilisée. Au survol d'une ligne, les rangées concernées sont mises en surbrillance sur le canvas.
