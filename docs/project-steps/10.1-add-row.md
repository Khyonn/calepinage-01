# Sous-étape 10.1 — Ajouter une rangée

## 1. Objectif utilisateur

Sur une pièce active en mode `edit`, l'utilisateur sélectionne un type de lame dans une combobox et clique "Ajouter une rangée". Une nouvelle rangée apparaît, positionnée en Y à la suite des rangées existantes. Chaque segment démarre avec `xOffset: 0`.

- Jalon 09 ✅ (pièces existent).
- Jalon 06.3 ✅ (catalogue).
- Store : `addRowThunk` existe déjà (étape 05).
