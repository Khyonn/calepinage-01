# Sous-étape 08.3 — Calibration

## 1. Objectif utilisateur

En mode `plan`, l'utilisateur place deux points A et B sur des repères identifiables du plan (ex. deux coins d'un mur de dimension connue), saisit la distance réelle en cm, et valide. L'application calcule automatiquement l'échelle pixels → cm, ajuste la taille de l'image pour que la distance A-B corresponde à la distance réelle en unités monde.

## 2. Dépendances

- 08.1 (plan importé), 08.2 (position stable).
- Domaine : type `Calibration` déjà présent dans `core/types.ts`.

## 3. Fichiers à créer / modifier

**Créer :**
- `src/components/canvas/BackgroundPlan/CalibrationPoints/index.tsx` — rendu SVG des 2 points A et B, draggables, avec labels "A" et "B". **Le visuel (forme, couleur, taille, décorations) reproduit le wireframe** — voir `docs/wireframe-iterations/` (chercher la frame calibration). Ne pas inventer un style divergent.
- `src/components/canvas/BackgroundPlan/CalibrationPoints/useCalibrationDrag.ts` — drag des deux points.
- `src/components/panels/CalibrationForm/index.tsx` — champ "Distance réelle" (cm) + boutons ✕ (annuler) et ✓ (valider).
- `src/core/calibration.ts` — fonction pure `applyCalibration(plan, calibration) → Plan` qui calcule le facteur d'échelle et retourne un plan "calibré" (ajuste les dimensions affichées de l'image via un scale appliqué dans le SVG). Peut se contenter de stocker `Calibration` et dériver le scale au rendu.

**Modifier :**
- `src/components/canvas/BackgroundPlan/index.tsx` — monter `<CalibrationPoints />` en mode `plan`.
- `src/components/panels/PlanImportForm/index.tsx` — ajouter un bouton "Calibrer" qui ouvre le `CalibrationForm`.
- `src/store/projectSlice.ts` — reducer `setCalibration(Calibration)`.

## 4. Comportement attendu

- [ ] Mode `plan` active, bouton "Calibrer" dans le panneau.
- [ ] Clic → les points A et B apparaissent au centre visible du canvas (positions initiales à 100 cm l'un de l'autre).
- [ ] Points draggables indépendamment avec snap visuel sur les vertices existants (optionnel — acceptable sans snap à ce stade).
- [ ] Champ "Distance réelle" avec validation : nombre > 0, précision 0,1 cm.
- [ ] ✓ → dispatch `setCalibration`, l'image est redimensionnée en conséquence. Les points disparaissent.
- [ ] ✕ → annule, points disparaissent sans dispatch.
- [ ] Si une calibration existe déjà, bouton "Calibrer" la réouvre pour modification (points à leurs anciennes positions).
- [ ] L'échelle calibrée est appliquée au rendu de l'image : la largeur visuelle de l'image en cm monde = largeur_pixels_image × (distance_réelle / distance_points_pixels).
