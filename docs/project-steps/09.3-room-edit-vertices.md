# Sous-étape 09.3 — Mode `edit` sans rangées : édition des sommets

## 1. Objectif utilisateur

Sur une pièce **sans rangées**, l'utilisateur peut drag ses sommets pour ajuster le contour. Avec `Ctrl`, un snap axial X+Y (même seuil `0.75 rem` écran qu'en dessin) aligne sur les autres sommets. `Shift` + drag pousse aussi les deux sommets adjacents ("pousser un mur"). `Échap` annule les modifications en cours.

## 2. Dépendances

- 09.1 (snap partagé via `src/core/snap.ts`), 09.2 (rendu Room).

## 3. Fichiers à créer / modifier

**Créer :**
- `src/components/canvas/Room/VertexHandles/index.tsx` — rendu des croix (✕) sur les sommets. Actif seulement si `mode === 'edit'` ET pièce active sans rangées.
- `src/components/canvas/Room/VertexHandles/useVertexDrag.ts` — drag individuel, avec snap (réutilise `findSnap`), Shift = push mur, Échap = rollback.

**Modifier :**
- `src/components/canvas/Room/index.tsx` — monter `<VertexHandles />` conditionnellement.
- `src/store/projectSlice.ts` — s'assurer que `updateRoomVertices` (ou `updateRoom({ vertices })`) existe et est dispatchable au release du drag (pas pendant — état local dans le hook, dispatch final).

## 4. Comportement attendu

- [ ] Pièce active sans rangées → croix visibles sur chaque vertex en mode `edit`.
- [ ] Pièce active avec rangées → pas de croix (édition des rangées prend le relais, jalon 10).
- [ ] Autres pièces → atténuées (voir 09.2).
- [ ] Drag d'un vertex → mise à jour locale en temps réel, dispatch au release.
- [ ] `Ctrl` maintenu → snap X+Y indépendants, lignes guides par axe.
- [ ] `Shift` maintenu → les deux vertices adjacents sont translatés du même delta (push mur).
- [ ] `Échap` pendant un drag → annule le drag, restaure la position initiale.
- [ ] Bascule hors mode `edit` → le drag en cours est annulé (rollback).

## 5. Tests

- **BDD** (`tests/features/room-edit-vertices.feature`) :
  - Scénario : "Je déplace un sommet d'une pièce sans rangées" — `selectRoomById(id).vertices` reflète la nouvelle position.
  - Scénario : "J'utilise Shift pour pousser un mur" — les deux sommets adjacents sont translatés.
  - Scénario : "Je ne peux pas éditer les sommets d'une pièce avec rangées" — dispatch ne passe pas (ou handle absent) — à valider UI plus que store.
