# Sous-étape 15.6 — yOffset par pièce

## 1. Objectif utilisateur

L'utilisateur peut décaler verticalement la première rangée d'une pièce via un champ « Décalage Y » (mm) dans le panneau d'édition pièce. Permet d'aligner les joints entre deux pièces adjacentes dont les bords de référence diffèrent (continuité visuelle linéaire).

## 5. Détails de calcul

**Formule centrale (`computeRowYStart`) :**
```
yStart = roomMinY + clampYOffset(yOffset, catalog) + cale + Σ(prior row widths)
```

**Clamp :**
```ts
clampYOffset(raw, catalog) =
  if raw > 0 → 0
  else if raw < -max(catalog.width) → -max
  else → raw
```

Appliqué :
- au write (slice `setRoomYOffset`) avec le catalogue courant ;
- au read défensif (`computeRowYStart`) pour absorber les changements de catalogue postérieurs.

## 8. Risques & arbitrages

- **Précision** : stockage en cm (cohérent avec les autres dimensions) ; UI en mm (granularité demandée).
- **Clamp runtime** : choix « lazy » (au calcul) plutôt qu'eager (synchronisation à chaque mutation du catalogue). Préserve l'intention : si l'utilisateur supprime puis ré-ajoute un type large, l'ancien yOffset redevient valide.
- **Pas de validation visuelle d'alignement** entre pièces voisines — fonctionnalité future éventuelle, hors scope.
- **Mode `nav` lecture seule** — non, `RoomEditPanel` est conditionné au mode `edit` ; pas de besoin d'affichage en mode `nav`.
