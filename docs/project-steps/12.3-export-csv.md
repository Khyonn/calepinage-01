# Sous-étape 12.3 — Export CSV ✅

## 1. Objectif utilisateur

Depuis le menu hamburger, l'utilisateur clique "Exporter CSV" et un fichier `.csv` est téléchargé. Il contient le résumé matière (par type de lame), les liens de réutilisation et les paramètres de pose. Ouvrable directement dans Excel / LibreOffice / Numbers.

## 4. Format CSV

Multi-section, séparateur `;` (compatible Excel FR), encodage UTF-8 + BOM, fin de ligne `\r\n` :

```
# Résumé matière
Type de lame;Quantité;Coût unitaire (€);Coût total (€)
Chêne 120×14 cm;12;45.00;540.00
...
Total;;;540.00

# Liens de réutilisation
Pièce source;Rangée source;Pièce cible;Rangée cible;Longueur réutilisée (cm)
Salon;2;Salon;4;38.0
...

# Paramètres de pose
Cale (cm);Largeur de scie (cm);Longueur min (cm);Écart rangées min (cm)
0.5;0.1;30.0;15.0
```

**Échappement** : si une cellule contient `;`, `"`, `\r` ou `\n` → entourée de `"…"`, les `"` internes doublés.

**Projet vide** : sections résumé/liens n'affichent que leur en-tête. Pas de ligne "Total" si aucun type n'a de lame commandée.

## 5. Nommage fichier

`calepinage-<slug>-<YYYY-MM-DD>.csv` où `<slug>` est dérivé du nom projet (NFD + suppression accents + remplacement non-alphanumériques par `-`, fallback `projet` si vide).
