# Spec 16.3 — Fondations canvas + URL

## Objectif

Session S3 : poser les fondations canvas/URL pour la suite de la refonte. Cinq chantiers indépendants livrés ensemble :

1. **Routing maison** History API (~100 LOC, pas de lib) — URL `/:projetId/:mode/:roomId?` synchronisée bi-directionnellement avec Redux.
2. **Topbar** réduite à 3 éléments (nom projet, sélecteur mode, menu actions).
3. **Theme grid** canvas (clair/sombre) via custom properties + `localStorage` direct, hook `useTheme`.
4. **Auto-fit** zoom/pan sur la pièce active au load et au changement de pièce.
5. **Migration drag** vers `@use-gesture/vanilla` : Scene (pan/zoom/pinch) + `useBackgroundPlanDrag` (pattern preview/commit, dispatch unique au release).

## Pré-requis

- Branche : `rewrite/v2`. Spec 16.2 mergée (couches `persistence/` propres, hooks `useTheme` / `useLastProjectId` co-localisés).
- Lire avant : `.agent/conventions.md`, `.agent/patterns/new-component.md`, `.agent/patterns/new-store-test.md`, `.agent/patterns/new-bdd-test.md`.

  - Pas de lib routing (`wouter`, `react-router`). Implémentation maison `src/router/`.
  - `@use-gesture/vanilla` (pas `/react`).

  - `localStorage` direct pour le thème (pas Redux, pas IDB).
  - `redux-undo` arrive en S6 → ne pas l'introduire ici.

## Détails par changement

### A. Routing maison — History API

#### A.1 Types et shape

`src/router/types.ts` :

```ts
export type RouteMode = 'main' | 'plan' | 'draw'

export interface RouteState {
  projectId: string | null
  mode: RouteMode
  roomId: string | null
}

export const DEFAULT_ROUTE: RouteState = {
  projectId: null, mode: 'main', roomId: null,
}
```

Convention URL :
- `/new` — pas de projet (création / liste vide).
- `/:projectId` — fallback automatique vers `/:projectId/main`.
- `/:projectId/:mode` — sans pièce.
- `/:projectId/:mode/:roomId` — pièce active.

Modes URL : `main` | `plan` | `draw`. `main` est l'alias URL de l'état Redux `nav` ou `edit` (la fusion arrive en S4 ; pour S3 on mappe explicitement, voir A.4).

#### A.2 `parseUrl` (fonction pure)

`src/router/parseUrl.ts` :

```ts
import type { ProjectsListEntry } from '@/store/types'
import type { RouteState, RouteMode } from './types'
import { DEFAULT_ROUTE } from './types'

const MODES: RouteMode[] = ['main', 'plan', 'draw']

export function parseUrl(pathname: string, list: ProjectsListEntry[]): RouteState {
  const segs = pathname.split('/').filter(Boolean)
  if (segs.length === 0 || segs[0] === 'new') return DEFAULT_ROUTE

  const [projectId, modeRaw, roomId] = segs
  const projectExists = list.some(p => p.id === projectId)
  if (!projectExists) return DEFAULT_ROUTE

  const mode: RouteMode = MODES.includes(modeRaw as RouteMode)
    ? (modeRaw as RouteMode)
    : 'main'

  return { projectId, mode, roomId: roomId ?? null }
}
```

Tests `src/router/__tests__/parse-url.feature` :
- `/new` → `DEFAULT_ROUTE`.
- `/` → `DEFAULT_ROUTE`.
- `/abc/main` avec projet `abc` listé → `{abc, main, null}`.
- `/abc` avec projet `abc` listé → `{abc, main, null}` (fallback mode).
- `/abc/draw/r1` → `{abc, draw, r1}`.
- `/abc/badmode/r1` → `{abc, main, r1}` (mode coercé).
- `/missing/main` projet non listé → `DEFAULT_ROUTE`.

#### A.3 `buildUrl` + `pushUrl`

`src/router/pushUrl.ts` :

```ts
import type { RouteState } from './types'

export function buildUrl(s: RouteState): string {
  if (!s.projectId) return '/new'
  const tail = s.roomId ? `/${s.mode}/${s.roomId}` : `/${s.mode}`
  return `/${s.projectId}${tail}`
}

export function pushUrl(url: string): void {
  if (window.location.pathname === url) return
  window.history.pushState(null, '', url)
}
```

Tests build (mêmes specs que parse, round-trip) :
- `DEFAULT_ROUTE` → `/new`.
- `{abc, main, null}` → `/abc/main`.
- `{abc, draw, r1}` → `/abc/draw/r1`.
- Round-trip : `parseUrl(buildUrl(s), [{id:s.projectId,...}]) === s` pour tout `s` valide.

#### A.4 `useRouterSync` — binding URL ↔ Redux

`src/hooks/useRouterSync.ts` :

```ts
import { useEffect } from 'react'
import { useAppDispatch, useAppSelector } from '@/hooks/redux'
import { uiActions } from '@/store/uiSlice'
import { selectProjectList, selectInteractionMode, selectActiveRoomId, selectCurrentProjectId } from '@/store/selectors'
import { openProjectThunk } from '@/store/thunks'
import { useLastProjectId } from '@/hooks/useLastProjectId'
import { parseUrl } from '@/router/parseUrl'
import { buildUrl, pushUrl } from '@/router/pushUrl'
import type { RouteMode } from '@/router/types'

const URL_TO_REDUX_MODE = (m: RouteMode) => m === 'main' ? 'nav' : m
const REDUX_TO_URL_MODE = (m: string): RouteMode =>
  m === 'plan' ? 'plan' : m === 'draw' ? 'draw' : 'main'

export function useRouterSync(): void {
  const dispatch = useAppDispatch()
  const list = useAppSelector(selectProjectList)
  const projectId = useAppSelector(selectCurrentProjectId)
  const mode = useAppSelector(selectInteractionMode)
  const roomId = useAppSelector(selectActiveRoomId)
  const lastStore = useLastProjectId()

  // URL → store (popstate + initial)
  useEffect(() => {
    const apply = () => {
      const route = parseUrl(window.location.pathname, list)
      if (route.projectId && route.projectId !== projectId) {
        void dispatch(openProjectThunk(route.projectId, lastStore))
      }
      dispatch(uiActions.setMode(URL_TO_REDUX_MODE(route.mode)))
      dispatch(uiActions.setActiveRoom(route.roomId))
    }
    apply()
    window.addEventListener('popstate', apply)
    return () => window.removeEventListener('popstate', apply)
  }, [dispatch, list, projectId, lastStore])

  // Store → URL
  useEffect(() => {
    const url = buildUrl({
      projectId, mode: REDUX_TO_URL_MODE(mode), roomId,
    })
    pushUrl(url)
  }, [projectId, mode, roomId])
}
```

Co-localisation : si le hook dépasse 100 LOC, splitter en `useUrlToStore.ts` + `useStoreToUrl.ts`.

### D. Auto-fit canvas

#### D.1 Logique pure `core/autoFit.ts`

```ts
import type { Point, Viewport } from '@/core/types'

interface AutoFitInput {
  vertices: Point[]
  viewportWidth: number
  viewportHeight: number
  marginRatio?: number
}

export function computeAutoFit({ vertices, viewportWidth, viewportHeight, marginRatio = 0.1 }: AutoFitInput): Viewport {
  if (vertices.length === 0 || viewportWidth === 0 || viewportHeight === 0) {
    return { zoom: 1, panX: viewportWidth / 2, panY: viewportHeight / 2 }
  }
  const xs = vertices.map(v => v.x), ys = vertices.map(v => v.y)
  const minX = Math.min(...xs), maxX = Math.max(...xs)
  const minY = Math.min(...ys), maxY = Math.max(...ys)
  const w = maxX - minX || 1, h = maxY - minY || 1
  const usableW = viewportWidth * (1 - 2 * marginRatio)
  const usableH = viewportHeight * (1 - 2 * marginRatio)
  const zoom = Math.min(usableW / w, usableH / h)
  const cx = (minX + maxX) / 2, cy = (minY + maxY) / 2
  return {
    zoom,
    panX: viewportWidth / 2 - cx * zoom,
    panY: viewportHeight / 2 - cy * zoom,
  }
}
```

Tests `core/__tests__/auto-fit.feature` (logique pure) :
- Pièce 100×100 dans viewport 1000×1000, marge 10 % → zoom = 8, pan centré.
- Pièce 200×100 dans viewport 1000×1000 → zoom dicté par la dimension dominante.
- Vertices vides → viewport identité centré.
- Viewport 0×0 → identité.

#### D.2 Hook `useAutoFit`

```ts
import { useEffect, useRef, type RefObject } from 'react'
import { useAppSelector } from '@/hooks/redux'
import { selectActiveRoom, selectRooms } from '@/store/selectors'
import type { ViewportApi } from '@/hooks/useViewport'
import { computeAutoFit } from '@/core/autoFit'

interface Args {
  svgRef: RefObject<SVGSVGElement | null>
  api: ViewportApi
  userPannedRef: RefObject<boolean>
}

export function useAutoFit({ svgRef, api, userPannedRef }: Args): void {
  const room = useAppSelector(selectActiveRoom)
  const rooms = useAppSelector(selectRooms)
  const setViewportRef = useRef(api.setViewport)
  setViewportRef.current = api.setViewport

  useEffect(() => {
    userPannedRef.current = false
    const svg = svgRef.current
    if (!svg) return
    const fit = () => {
      const r = svg.getBoundingClientRect()
      const vertices = room?.vertices ?? rooms.flatMap(r => r.vertices)
      setViewportRef.current(computeAutoFit({
        vertices, viewportWidth: r.width, viewportHeight: r.height,
      }))
    }
    fit()
    const ro = new ResizeObserver(() => { if (!userPannedRef.current) fit() })
    ro.observe(svg)
    return () => ro.disconnect()
  }, [room?.id, rooms, userPannedRef])
}
```

**Coordination** : `userPannedRef` partagé entre `useAutoFit` et `useCanvasEvents` via `Scene` :

```ts
const userPannedRef = useRef(false)
useAutoFit({ svgRef, api, userPannedRef })
useCanvasEvents(svgRef, api, userPannedRef)
```

`useAutoFit` reset le ref à chaque changement de `room?.id`. Au premier `onDrag` non programmatique, `useCanvasEvents` set `userPannedRef.current = true`.

#### D.3 Trigger

L'effet de `useAutoFit` se relance à chaque changement de `room?.id` ou de la liste `rooms`. Le `ResizeObserver` ré-applique le fit uniquement si l'utilisateur n'a pas pané.

#### E.2 `useCanvasEvents` réécrit

```ts
import { useEffect, useRef, useState, type RefObject } from 'react'
import { Gesture } from '@use-gesture/vanilla'
import type { ViewportApi } from '@/hooks/useViewport'
import { useSpaceKey } from '@/hooks/useSpaceKey'

export type CursorMode = '' | 'spaceDown' | 'panning'

export function useCanvasEvents(
  svgRef: RefObject<SVGSVGElement | null>,
  api: ViewportApi,
  userPannedRef: RefObject<boolean>,
): { cursorMode: CursorMode } {
  const [isPanning, setIsPanning] = useState(false)
  const { spaceDownRef, spaceDown } = useSpaceKey()
  const apiRef = useRef(api); apiRef.current = api

  useEffect(() => {
    const svg = svgRef.current
    if (!svg) return
    const gesture = new Gesture(svg, {
      onWheel: ({ event, delta: [dx, dy], ctrlKey, shiftKey }) => {
        event.preventDefault?.()
        if (ctrlKey) {
          const factor = dy < 0 ? 1.1 : 1 / 1.1
          apiRef.current.zoomAt(
            (event as WheelEvent).offsetX, (event as WheelEvent).offsetY, factor,
          )
        } else if (shiftKey) {
          apiRef.current.pan(-dy, 0)
        } else {
          apiRef.current.pan(-dx, -dy)
        }
      },
      onDrag: ({ first, last, delta: [dx, dy], buttons }) => {
        const middle = (buttons & 4) === 4
        const allow = spaceDownRef.current || middle
        if (!allow) return
        if (first) setIsPanning(true)
        apiRef.current.pan(dx, dy)
        userPannedRef.current = true
        if (last) setIsPanning(false)
      },
      onPinch: ({ origin: [ox, oy], offset: [scale], memo }) => {
        const prev = (memo as number | undefined) ?? scale
        const factor = scale / prev
        const rect = svg.getBoundingClientRect()
        apiRef.current.zoomAt(ox - rect.left, oy - rect.top, factor)
        return scale
      },
    }, {
      drag: { pointer: { buttons: [1, 4] } },
      eventOptions: { passive: false },
    })
    return () => gesture.destroy()
  }, [svgRef, spaceDownRef, userPannedRef])

  const cursorMode: CursorMode = isPanning ? 'panning' : spaceDown ? 'spaceDown' : ''
  return { cursorMode }
}
```

Notes :
- Vérifier la version installée pour la signature des handlers (`delta` vs `movement`).
- Le pan reste local React (`setViewport`), pas de dispatch Redux. OK avec la règle preview/commit.
- Conserver fallback molette milieu (`buttons & 4`) en parallèle de espace + clic gauche.
- `event.preventDefault()` requiert `passive: false`.

### F. `useBackgroundPlanDrag` — preview / commit

```ts
  useEffect(() => {
    const img = imageRef.current
    if (!img || !plan || mode !== 'plan') return
    const gesture = new Gesture(img, {
      onDrag: ({ first, last, movement: [mx, my], cancel, buttons, event }) => {
        if (spaceDownRef.current || (buttons & 1) !== 1) { cancel(); return }
        event.stopPropagation()
        if (first) startRef.current = { x: plan.x, y: plan.y }
        const start = startRef.current
        if (!start) return
        const z = zoomRef.current || 1
        const next = { x: start.x + mx / z, y: start.y + my / z }
        if (last) {
          dispatch(projectActions.updatePlan(next))
          setDraft(null)
          startRef.current = null
        } else {
          setDraft(next)
        }
      },
    }, { eventOptions: { passive: false } })
    return () => gesture.destroy()
  }, [imageRef, plan, mode, dispatch, spaceDownRef])

  return { draft }
}
```

Garanties :
- **Aucun dispatch** Redux pendant le drag.
- **Un seul** `dispatch(updatePlan)` au release.
- Le composant consommateur lit `draft` et l'applique en `transform` SVG si présent.

## Rollback / risques

- **Risque `@use-gesture/vanilla`** : dépendance nouvelle, vérifier compat Vite 8 + React 19. Fallback : revert à pointer-events précédent.
- **Risque popstate boucle** : guard `if (window.location.pathname === url) return` dans `pushUrl`.
- **Risque `userPannedRef`** : ref partagé inhabituel ; si bug visuel, remonter dans store (`uiSlice.userPannedSinceLastFit`).
- **Risque CSS custom props dans SVG** : préférer classes CSS module, pas `stroke="var(...)"` direct.
- **Risque double load** : s'assurer que `parseUrl` voit la `list` non-vide au premier render. `preloadedState` du store doit contenir `list` rempli **avant** le mount React.
