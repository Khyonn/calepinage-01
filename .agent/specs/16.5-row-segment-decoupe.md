# Spec 16.5 — Découpe Row/Segment + drags @use-gesture

## Contexte & objectif

Deux composants de `src/components/canvas/Row/` violent les conventions et freinent les jalons suivants (undo S6) :

1. `Row/index.tsx` (~142 LOC, 8 hooks Redux : 7 `useAppSelector` + 1 `useAppDispatch`, plus `useViewportContext`, `useRowGeometry`, `useSegmentDrag`). Cible : ≤ 3 hooks Redux.
2. `Row/Segment/index.tsx` (151 LOC, 19 props). Cible : ≤ 8 props et ≤ 150 LOC.

Deux hooks de drag (`useSegmentDrag`, `useVertexDrag`) branchent listeners `pointermove`/`pointerup` à la main (~70 LOC chacun). Pour `useVertexDrag`, dispatch dans `onUp` uniquement (déjà conforme preview/commit). Pour `useSegmentDrag`, idem. Migration sur `@use-gesture/vanilla` aligne avec `useBackgroundPlanDrag` (S3) et réduit la dette.

Cette session :
- A. Découpe `Row/index.tsx` en orchestrateur fin (≤ 3 hooks Redux) + sous-composants pures.
- B. Découpe `Row/Segment/index.tsx` en groupant les 19 props en deux objets (view-model + handlers) + extraction de 2-3 sous-composants.
- C. Migre `useSegmentDrag` et `useVertexDrag` sur `@use-gesture/vanilla`.
- D. Ajoute sélecteur composé `selectRowViewModel` factory pour remplacer les 7 sélecteurs individuels.

## Pré-requis

- Branche : `rewrite/v2`.
- Lire avant : `.agent/conventions.md`, `.agent/patterns/new-component.md`, `.agent/patterns/new-store-test.md`.
- `@use-gesture/vanilla` introduit en S3. Vérifier `package.json`.
- Pas de changement IDB. Pas de changement schéma JSON. Pas de changement d'actions Redux (`updateSegmentOffset` et `updateRoom` existent et restent les **uniques** actions dispatchées par les drags).

#### `Row/index.tsx` cible

```tsx
import { useMemo } from 'react'
import { useAppDispatch, useAppSelector } from '@/hooks/redux'
import { makeSelectRowViewModel } from '@/store/selectors'
import { useViewportContext } from '@/components/canvas/Scene/ViewportContext'
import { fillRow } from '@/core/rowFill'
import { computeLiveRowViolations } from './liveRowViolations'
import { useRowGeometry } from './useRowGeometry'
import { useSegmentDrag } from './useSegmentDrag'
import { useRowDispatchers } from './useRowDispatchers'
import { RowSegments } from './RowSegments'

interface Props { roomId: string; rowId: string }

export function Row({ roomId, rowId }: Props) {
  const dispatch = useAppDispatch()
  const geometry = useRowGeometry(roomId, rowId)
  const selectVM = useMemo(makeSelectRowViewModel, [])
  const vm = useAppSelector(s => selectVM(s, { roomId, rowId }))
  const { viewport } = useViewportContext()

  const initialXOffsets = useMemo(
    () => vm?.segments.map(s => s.xOffset) ?? [],
    [vm],
  )

  const { previewOffsets, draggingIndex, onSegmentRef } = useSegmentDrag({
    roomId, rowId,
    active: vm?.dragActive ?? false,
    initialXOffsets,
    plankLength: geometry?.plankType.length ?? 0,
  })

  const commit = useRowDispatchers({ dispatch, roomId, rowId, geometry, poseParams: vm?.poseParams ?? null })

  if (!geometry || !vm) return null

  const effectiveSegments = previewOffsets === null
    ? geometry.segments
    : geometry.segments.map((seg, i) => previewOffsets[i] === initialXOffsets[i]
        ? seg
        : { ...seg, planks: fillRow(previewOffsets[i], seg.xEnd - seg.xStart, geometry.plankType, vm.poseParams) })

  const isDragging = draggingIndex !== null
  const rowViolations = isDragging
    ? computeLiveRowViolations(vm.project, roomId, rowId, effectiveSegments, vm.poseParams)
    : vm.allViolations.filter(v => v.rowId === rowId)

  return (
    <RowSegments
      roomId={roomId}
      geometry={geometry}
      effectiveSegments={effectiveSegments}
      snapshotSegments={isDragging ? geometry.segments : undefined}
      poseParams={vm.poseParams}
      zoom={viewport.zoom}
      startLinked={vm.startLinked}
      endLinked={vm.endLinked}
      rowViolations={rowViolations}
      dragActive={vm.dragActive}
      draggingIndex={draggingIndex}
      highlightFirst={vm.highlightFirst}
      highlightLast={vm.highlightLast}
      onSegmentRef={onSegmentRef}
      onCommitFirstLength={commit.firstLength}
      onCommitLastLength={commit.lastLength}
    />
  )
}
```

**Décompte hooks Redux** : 1 `useAppDispatch` + 1 `useAppSelector` = **2** ≤ 3. ✓
LOC cible : ~55 ≤ 150. ✓

#### `useRowDispatchers.ts`

```ts
import { useMemo } from 'react'
import type { Dispatch } from '@reduxjs/toolkit'
import type { PoseParams } from '@/core/types'
import type { RowGeometry } from '@/core/rowGeometry'
import { projectActions } from '@/store/projectSlice'
import { xOffsetFromFirstLength, xOffsetFromLastLength } from '@/core/xOffsetFromPlankLength'

interface Args {
  dispatch: Dispatch
  roomId: string
  rowId: string
  geometry: RowGeometry | null
  poseParams: PoseParams | null
}

export function useRowDispatchers({ dispatch, roomId, rowId, geometry, poseParams }: Args) {
  return useMemo(() => ({
    firstLength: (segmentIndex: number, newLength: number) => {
      if (!geometry) return
      const x = xOffsetFromFirstLength(newLength, geometry.plankType)
      dispatch(projectActions.updateSegmentOffset({ roomId, rowId, segmentIndex, xOffset: x }))
    },
    lastLength: (segmentIndex: number, newLength: number) => {
      if (!geometry || !poseParams) return
      const seg = geometry.segments[segmentIndex]
      if (!seg) return
      const segWidth = seg.xEnd - seg.xStart
      const x = xOffsetFromLastLength(newLength, geometry.plankType, segWidth, poseParams)
      dispatch(projectActions.updateSegmentOffset({ roomId, rowId, segmentIndex, xOffset: x }))
    },
  }), [dispatch, roomId, rowId, geometry, poseParams])
}
```

```ts
import type { Plank, PoseParams, ConstraintViolation } from '@/core/types'
import { messageForViolation } from './ViolationBadge'

const round1 = (n: number) => Math.round(n * 10) / 10

export function computeSegmentMessages(
  planks: Plank[], poseParams: PoseParams, rowViolations: ConstraintViolation[],
): string[] {
  const minLen = poseParams.minPlankLength
  const first = planks[0]
  const last = planks[planks.length - 1]
  const out: string[] = []
  if (first && first.length < minLen) {
    out.push(messageForViolation('first-plank-too-short', { length: round1(first.length), min: minLen }))
  }
  if (last && planks.length > 1 && last.length < minLen) {
    out.push(messageForViolation('last-plank-too-short', { length: round1(last.length), min: minLen }))
  }
  const gap = rowViolations.find(v => v.type === 'row-gap-too-small')
  if (gap) {
    out.push(messageForViolation('row-gap-too-small', { gap: round1(gap.value), min: poseParams.minRowGap }))
  }
  return out
}
```

#### `useSegmentDrag` — après

```ts
import { Gesture } from '@use-gesture/vanilla'
import { useCallback, useEffect, useRef, useState } from 'react'
import type { RefCallback } from 'react'
import { useAppDispatch } from '@/hooks/redux'
import { projectActions } from '@/store/projectSlice'
import { useViewportContext } from '@/components/canvas/Scene/ViewportContext'

interface Params {
  roomId: string; rowId: string
  active: boolean
  initialXOffsets: number[]
  plankLength: number
}

export function useSegmentDrag({ roomId, rowId, active, initialXOffsets, plankLength }: Params) {
  const dispatch = useAppDispatch()
  const { viewport } = useViewportContext()
  const [previewOffsets, setPreviewOffsets] = useState<number[] | null>(null)
  const [draggingIndex, setDraggingIndex] = useState<number | null>(null)

  const refs = useRef<Record<number, SVGGElement | null>>({})
  const gestures = useRef<Record<number, Gesture>>({})

  useEffect(() => {
    Object.values(gestures.current).forEach(g => g.destroy())
    gestures.current = {}
    if (!active) return

    const quantize = (n: number) => Math.round(n * 10) / 10
    const maxOffset = Math.max(0, plankLength - 0.1)
    const startZoom = viewport.zoom

    Object.entries(refs.current).forEach(([key, el]) => {
      if (!el) return
      const segmentIndex = Number(key)
      const initial = initialXOffsets[segmentIndex] ?? 0
      gestures.current[segmentIndex] = new Gesture(el, {
        onDrag: ({ first, last, canceled, movement: [dx], event }) => {
          event.stopPropagation()
          if (first) setDraggingIndex(segmentIndex)
          const deltaWorld = dx / startZoom
          const raw = quantize(initial - deltaWorld)
          const next = Math.max(0, Math.min(maxOffset, raw))
          const arr = [...initialXOffsets]
          arr[segmentIndex] = next
          if (canceled) {
            setPreviewOffsets(null); setDraggingIndex(null); return
          }
          if (last) {
            dispatch(projectActions.updateSegmentOffset({ roomId, rowId, segmentIndex, xOffset: next }))
            setPreviewOffsets(null); setDraggingIndex(null)
          } else {
            setPreviewOffsets(arr)
          }
        },
      }, { drag: { filterTaps: true, pointer: { keys: false } } })
    })

    return () => {
      Object.values(gestures.current).forEach(g => g.destroy())
      gestures.current = {}
    }
  }, [active, plankLength, initialXOffsets, viewport.zoom, dispatch, roomId, rowId])

  const onSegmentRef = useCallback((segmentIndex: number): RefCallback<SVGGElement> => (el) => {
    refs.current[segmentIndex] = el
  }, [])

  return { previewOffsets, draggingIndex, onSegmentRef }
}
```

**Notes** :
- API publique : `onSegmentRef` (callback factory de ref) au lieu de `onSegmentPointerDown`.
- `Gesture` gère pointer-capture, `pointercancel`, et cancel via Escape (vérifier doc à jour).
- Dispatch **uniquement** dans `last: true` → conforme preview/commit.

#### `useVertexDrag` — après

```ts
import { Gesture } from '@use-gesture/vanilla'
import { useCallback, useEffect, useRef, useState } from 'react'
import type { Point } from '@/core/types'
import { findSnap } from '@/core/snap'
import { useAppDispatch } from '@/hooks/redux'
import { projectActions } from '@/store/projectSlice'
import { useViewportContext } from '@/components/canvas/Scene/ViewportContext'
import { useVertexEdit } from '../VertexEditContext'

interface Params {
  roomId: string
  vertices: Point[]
  active: boolean
}

export function useVertexDrag({ roomId, vertices, active }: Params) {
  const dispatch = useAppDispatch()
  const { viewport, screenToWorld, svgRef } = useViewportContext()
  const { setDraft, clearDraft } = useVertexEdit()
  const [draggingIndex, setDraggingIndex] = useState<number | null>(null)

  const refs = useRef<Record<number, SVGGElement | null>>({})
  const gestures = useRef<Record<number, Gesture>>({})
  const rootFontPx = typeof window !== 'undefined'
    ? parseFloat(getComputedStyle(document.documentElement).fontSize) || 16
    : 16

  useEffect(() => {
    Object.values(gestures.current).forEach(g => g.destroy())
    gestures.current = {}
    if (!active) { clearDraft(); setDraggingIndex(null); return }
    const svg = svgRef.current
    if (!svg) return
    const startVertices = vertices.map(v => ({ ...v }))

    const buildNext = (index: number, world: Point, ctrl: boolean, shift: boolean) => {
      const others = startVertices.filter((_, i) => i !== index)
      const target = ctrl ? findSnap(world, others, viewport.zoom, rootFontPx) : world
      const dx = target.x - startVertices[index].x
      const dy = target.y - startVertices[index].y
      const n = startVertices.length
      return startVertices.map((v, i) => {
        if (i === index) return { x: target.x, y: target.y }
        if (shift && (i === (index - 1 + n) % n || i === (index + 1) % n)) {
          return { x: v.x + dx, y: v.y + dy }
        }
        return v
      })
    }

    Object.entries(refs.current).forEach(([key, el]) => {
      if (!el) return
      const index = Number(key)
      gestures.current[index] = new Gesture(el, {
        onDrag: ({ first, last, canceled, xy: [cx, cy], ctrlKey, shiftKey, event }) => {
          event.stopPropagation()
          const rect = svg.getBoundingClientRect()
          const world = screenToWorld({ x: cx - rect.left, y: cy - rect.top })
          if (first) setDraggingIndex(index)
          const next = buildNext(index, world, ctrlKey, shiftKey)
          if (canceled) { clearDraft(); setDraggingIndex(null); return }
          if (last) {
            dispatch(projectActions.updateRoom({ id: roomId, vertices: next }))
            clearDraft(); setDraggingIndex(null)
          } else setDraft(roomId, next)
        },
      }, { drag: { filterTaps: true } })
    })

    return () => {
      Object.values(gestures.current).forEach(g => g.destroy())
      gestures.current = {}
    }
  }, [active, vertices, svgRef, screenToWorld, viewport.zoom, rootFontPx, roomId, dispatch, setDraft, clearDraft])

  const onHandleRef = useCallback((index: number) => (el: SVGGElement | null) => {
    refs.current[index] = el
  }, [])

  return { onHandleRef, draggingIndex }
}
```

**Adapter `VertexHandles/index.tsx`** : `ref={onHandleRef(i)}` au lieu de `onPointerDown={e => startDrag(e, i)}`.

#### Bénéfices migration

- ~70 LOC boilerplate listeners supprimées par hook (× 2 = ~140 LOC).
- Annulation (`canceled` / Escape) gérée nativement.
- Filtrage tap (`filterTaps`) géré.
- Cleanup unifié (`gesture.destroy()`).
- Pattern uniforme avec `useBackgroundPlanDrag` (S3).

```ts
        project,
        segments: row.segments.map(s => ({ xOffset: s.xOffset })),
        startLinked: links.some(l => l.targetRowId === rowId),
        endLinked: links.some(l => l.sourceRowId === rowId),
        highlightFirst: highlights.some(h => h.rowId === rowId && h.position === 'first'),
        highlightLast: highlights.some(h => h.rowId === rowId && h.position === 'last'),
        dragActive: activeRoomId === roomId && mode === 'main',
        allViolations: violations,
      }
    },
  )
}
```

**Factory** obligatoire (cf. pattern `makeSelectRowGeometry`). Plusieurs `<Row>` partagent la signature → instances séparées via `useMemo(makeSelectRowViewModel, [])` côté composant. Sinon le cache `createSelector` s'invalide entre instances.

## Rollback / risques

- **Risque API @use-gesture** : si Escape pas capturé, listener `keydown` global dans le `useEffect`.
- **Risque ref vs onPointerDown** : poser ref sur `<g>` SVG OK en React 19. Vérifier qu'aucun parent n'écoute déjà `onPointerDown` et bloque propagation.
- **Risque sélecteur composé instable** : **toujours** factory + `useMemo` côté composant.
- **Risque API publique cassée** : `useSegmentDrag` retourne `onSegmentRef` au lieu de `onSegmentPointerDown`. `useVertexDrag` retourne `onHandleRef`. Grep usage : un seul consommateur chacun.
- **Ordre suggéré** : D (sélecteur) → A (Row) → B (Segment) → C segment → C vertex → tests.
